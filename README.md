# George Jarrell Handyman Services Website

A modern, responsive static website for George Jarrell's handyman business.

## Features

- **Single-page design** with smooth scrolling navigation
- **Responsive layout** that works on all devices
- **Modern UI** with gradient backgrounds and floating animations
- **Service showcase** with grid layout
- **Customer testimonials** section
- **Contact form** with email functionality
- **Professional styling** with consistent branding

## Structure

```
├── www/                   # Static website files
│   ├── css/
│   │   └── style.css      # Main stylesheet
│   ├── img/               # Image assets (placeholder files)
│   ├── js/
│   │   └── main.js        # JavaScript functionality
│   └── index.html         # Main website file
├── docs/                  # Documentation
└── README.md             # This file
```

## Sections

1. **Hero/Home** - Introduction with floating team images
2. **Services** - Grid of handyman services offered
3. **Testimonials** - Customer reviews and feedback
4. **About** - Information about George and the business
5. **Contact** - Contact details and email form

## Setup

1. Replace placeholder images in `img/` folder with actual photos
2. Update contact information in `index.html`
3. Customize colors and branding in `css/style.css`
4. Host on any static web hosting service

## Domain

Designed for: **handymanjarrell.com**

## Notes

- Built as a static website (HTML/CSS/JS only)
- No backend or database required
- Optimized for fast loading and SEO
- Ready for deployment to static hosting services
